| 运算符                                                                       | 说明                         |
|------------------------------------------------------------------------------|------------------------------|
| `()`、`[]`、`->` 、`.`                                                       | 圆括号、方括号、指针、成员   |
| `++` 、 `--` 、 `*` 、 `&` 、 `~` 、 `!` 、 `+` 、 `-` 、 `sizeof`、`(type)` | 单目运算符（右结合）         |
| `*` 、 `/` 、 `%`                                                            | 算术运算符                   |
| `+` 、`-`                                                                    | 算术运算符                   |
| `<<` 、 `>>`                                                                 | 位运算符                     |
| `<` 、 `<=` 、 `>` 、 `>=`                                                   | 关系运算符                   |
| `==` 、 `!=`                                                                 | 关系运算符                   |
| `&`                                                                          | 位与                         |
| `^`                                                                          | 位异或                       |
| `\|`                                                                         | 位或                         |
| `&&`                                                                         | 逻辑与                       |
| `\|\|`                                                                       | 逻辑或                       |
| `?` 、 `:`                                                                   | 条件运算符                   |
| `=` 、 `+=` 、 `-=` 、 `*=` 、 `/=` 、 `%=` 、 `&=` 、`\|=` 、 `^=`          | 赋值运算符（右结合）         |
